Descrição detalhada

Classe responsável pela persistência permanente de dados do simulador. Gerencia a gravação e leitura da lista global de setups em arquivos binários, garantindo o ciclo de vida dos dados entre diferentes sessões do programa. Aplica o Princípio da Responsabilidade Única (SRP).

◆ carregarSetups()

static List< Setup > pcbuilder.GerenciadorDados.carregarSetups ( ) throws IOException, ClassNotFoundException
static

Recupera a lista de setups previamente persistida em disco. Caso o arquivo não exista (como na primeira execução do sistema), inicializa silenciosamente uma nova lista vazia, evitando falhas de inicialização.

Retorna
List contendo as instâncias de Setup recriadas ou lista vazia.
Excepções
IOExceptionCaso ocorra erro de leitura ou corrupção do arquivo.
ClassNotFoundExceptionLançada se a JVM não conseguir mapear o tipo do objeto recuperado.
