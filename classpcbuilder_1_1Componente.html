Detailed Description

Classe abstrata base para todos os hardwares do sistema. Centraliza os atributos comuns como marca, modelo, preço e consumo elétrico, garantindo o reaproveitamento de código através da Herança. Implementa Serializable para permitir a persistência em disco de todas as subclasses.
